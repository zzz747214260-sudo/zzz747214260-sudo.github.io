Proposal: Personal Design Portfolio Website

Project Overview

This project aims to create a personal design portfolio website to showcase my works in visual communication and service design. The website will serve as a professional tool for job applications and as a long-term platform to document my design growth. It will present projects with clear visuals, explanations of design processes, and highlight my personal design philosophy.

Objectives

  • Showcase Design Works: Present projects in organized categories (e.g., service design, visual communication, research projects).
  • Professional Branding: Build a strong personal online identity emphasizing creativity and design thinking.
  • Ease of Updating: Ensure the website structure allows continuous addition of new projects and reflections.
  • Job Application Support: Provide recruiters and collaborators with easy access to my works, CV, and contact information.

Target Audience

  • Potential Employers in design studios, tech companies, and cultural institutions.
  • Academic Institutions evaluating applications for further study.
  • Peers and collaborators in the design community.

Features & Content

  • Homepage: Introduction, featured projects, and personal branding statement.
  • Portfolio Section: Galleries of projects with images, process explanations, and outcomes.
  • About Me: Biography, educational background, design philosophy, and CV link.
  • Blog / Updates: Reflections, news, or ongoing design explorations.
  • Contact Page: Email, LinkedIn, and other professional links.

Visualization Opportunities

  • Project Data: Pie charts showing project types or skill usage, timelines for project completion.
  • User Research: Bar charts, scatter plots, or heatmaps to display survey results and user behaviors.
  • Interactive Displays: Dynamic visualizations for exploring design processes or comparing project iterations.
  • Personal Growth: Line graphs or radar charts showing skill development over time.

Technical Approach

  • Phase 1 (Quick Start): Use template-based platforms like Wix, Squarespace, or WordPress for a professional-looking website.
  • Phase 2 (Custom Design): Learn HTML, CSS, JavaScript, and D3.js to implement interactive and personalized visualizations.
  • Ensure responsive design across devices and optional integration of interactive data visualizations.

Design Style

Minimalist layout to emphasize works. Consistent typography and color palette reflecting personal branding. Balance between visuals and concise explanatory texts.

Timeline

  • Weeks 1–2: Research, content organization, and project selection.
  • Weeks 3–4: Build initial website using templates.
  • Month 2+: Add custom interactions and visualizations, refine design.
  • Ongoing: Update with new projects and reflections.

Expected Outcomes

  • A professional, navigable portfolio website highlighting my design skills.
  • Enhanced personal branding for career opportunities.
  • A sustainable platform for documenting design explorations and growth.
